# Terraform AWS VPC Module

This Terraform module provisions a customizable Virtual Private Cloud (VPC) on AWS.

## 🚀 Features

- Creates a VPC with customizable CIDR block
- Supports public and private subnets
- Provisions Internet Gateway (IGW)
- Creates NAT Gateway for private subnet internet access
- Configures route tables and route associations
- Supports multiple availability zones
